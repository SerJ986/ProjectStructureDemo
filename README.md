# ProjectStructureDemo

Пример структуры проекта на C#

## Структура проекта

- **Models** - Классы данных
- **Services** - Бизнес-логика
- **Data** - Работа с данными
- **Utils** - Вспомогательные классы
- **Tests** - Модульные тесты

## Запуск проекта

1. Установите .NET 6+
2. Выполните `dotnet run`
